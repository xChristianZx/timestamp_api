Timestamp Microservice

Usage

Add unix time or natural date at the end of the root url

localhost:3000/ your time stamp here

Example:

url/1511251200
url/November 21, 2017

This will return

{unixTime": 1511251200, "naturalTime": "November 21, 2017",}

Note: If timestamp or date provided is not a provided, both values will return null
